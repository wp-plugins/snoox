Snoox Widgets Plugin allows site owners to embed Snoox Recommendations in their sites/blogs

== Description ==

Snoox Widgets Plugin allows site owners to embed Snoox Recommendations in their sites/blogs, you can embed a particular display from Snoox.com or a general recommendations list for particular subjects, e.g. Movies, Music, etc, Snoox.com is a social recommendation platform that allows users to browse thousands of real recommendations from their friends and the community in general, as a user you can also recommend to your friends/community about anything you like and love

== Installation ==

1. Upload `SnooxPlugin Folder` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to https://www.snoox.com/widgets/api to learn how to add widgets to your posts
